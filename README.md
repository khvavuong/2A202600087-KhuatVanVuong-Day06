## Getting Started

1. Set up a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Linux venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -e .
   ```
3. Set environment variables:
   Copy `.env.example` to `.env` and fill in your API keys (e.g., OPENAI_API_KEY).
4. Run the chatbot:
   ```bash
   langgraph dev
   ```
   -> Enjoy it
